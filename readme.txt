### Key Features
* Simple, light-weight, and super-fast
* <b>Built with only Gutenberg Native Components</b>
* <b>No Block Builder at all</b>
* <b>100% Responsive on different sizes of devices</b>

== Installation ==

This section describes how to install the plugin and get it working.

1. You can install the plugin from your WordPress Plugin menu.
2. Alternatively, you can put plugin's files to your WordPress plugin folder, i.e., `/wp-content/plugins/wtr-box-slogan-block` folder.
3. Activate the plugin through the 'Plugins" screen in WordPress.
4. Go to Gutenberg Block Editor and you will find a new category with a new Block named "Box Slogan Block".
5. Click on it to drop the "Box Slogan Block" on your Gutenberg editor.
